# Skill: Report Generator
Role: Agent 01 (OC Monitoring Agent)

## Description
Creates clean Markdown summaries with task counts, costs, and uptime statistics.

## Implementation Details
1. Aggregate data from Cost Tracker and Health Check.
2. Generate daily/weekly/monthly PDF and Markdown reports.
3. Save reports to `outputs/reports/`.
4. Send report download links via Notifier.
