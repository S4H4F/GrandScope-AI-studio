# Skill: SEO Optimizer
Role: Agent 02 (OC Content Writer)

## Description
Ensures all content follows strict SEO guidelines: keyword density 1-3%, meta descriptions <160 chars, and proper H1/H2 structure.

## Rules
1. Every piece of content must have exactly one H1 tag.
2. Primary keyword must appear in the first 100 words.
3. Subheadings (H2, H3) must include semantic variations of the primary keyword.
4. Meta description must be provided as YAML frontmatter.
